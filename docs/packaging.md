# Packaging

The Windows installer is generated by Electron Forge Squirrel.

Commands:

```powershell
npm install
npm run typecheck
npm test
npm run make
```

Expected output:

```text
out\make\squirrel.windows\x64\Codex Usage Setup.exe
out\make\squirrel.windows\x64\CodexUsage-1.0.0-full.nupkg
out\make\squirrel.windows\x64\RELEASES
```

Notes:

- The installer is unsigned. Windows SmartScreen may warn until the app is signed and reputation is established.
- Native SQLite packaging depends on `better-sqlite3` being rebuilt for Electron. Forge's auto-unpack plugin is enabled.
- Runtime data resolves through platform directories and does not depend on the source project path.
